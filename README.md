# 🏆 序位法評分平台 (Scoring Platform)

這是一個專為各類序位法評分統計的 Web 應用程式，採用**序位法**結合**加權總分**來進行公平、客觀的評分與排名結算。本平台讓管理者可以輕鬆建立評分事件、設定客製化評分項目，並一鍵產出委員專屬的評分連結。

## ✨ 核心特色
- **完善的管理員權限**：具備管理員登入機制 (`/login/`)，只有核可的管理員可以建立、管理、刪除評分事件，並查閱整體追蹤儀表板。
- **免註冊委員評分 (代碼入場)**：委員不需註冊帳號，只需透過管理員給予的單一「事件連結與代碼」，即可進入個人專屬評分介面。
- **動態自訂名單與匿名保護**：支援管理員逐筆輸入評選委員真實姓名；並提供「匿名模式選項」，一旦勾選，統計與報表自動切換顯示「委員 1、委員 2」，保障評閱隱私。
- **動態評分項目**：可為每次賽事設定多個評分項目，並自訂權重 (總和需為 100%)。
- **強制唯一序位**：確保每位委員給予各組的「序位」不重複（不可並列）。
- **自動化成績結算**：內建「序位總和越小者名次越高，序位相同時比較平均加權總分」的卓越競賽專業排名邏輯。
- **詳盡報表匯出**：可一鍵下載包含「總表」與「各委員評分明細」的多工作表 Excel (.xlsx) 檔案。
- **響應式 UI 設計**：採用現代化 Poppins 字體，以 #5B82F6 作為品牌藍色系，提供視覺乾淨流暢的操作體驗。

## 🛠️ 技術架構
- **後端框架**：Django 6.0 (Python)
- **資料庫**：SQLite3 (預設)
- **前端呈現**：HTML5, CSS3, JavaScript (Vanilla)
- **數據視覺化**：Chart.js
- **報表匯出**：openpyxl

## 🚀 快速啟動 (Local Development)

### 1. 準備環境
確保您已安裝 Python 3.10 以上版本。

### 2. 安裝套件
使用虛擬環境（建議）並安裝必要的 Python 套件：
```bash
# 建立並啟動虛擬環境 (Windows)
python -m venv venv
venv\Scripts\activate

# 安裝相依套件
pip install -r requirements.txt
```

### 3. 資料庫遷移
建立資料庫與資料表結構：
```bash
python manage.py makemigrations
python manage.py migrate
```

### 4. 啟動伺服器
```bash
python manage.py runserver
```
完成後，開啟瀏覽器造訪 `http://127.0.0.1:8000/` 即可看到平台首頁。

## 📖 平台使用流程
1. **管理員登入**：點擊首頁「管理員登入」，使用預設帳號（`admin`）登入。
2. **建立事件**：進入管理後台點擊「建立事件」，輸入自訂的獨立事件名稱與委員姓名清單（也可選擇使用匿名報表）。
3. **分享連結**：建立完畢後，在儀表板會產生「統一入場連結」與「進入代碼」，複製發送給所有委員。
4. **委員評分**：委員點擊統一連結，輸入代碼後選擇自己的名字即可評分。評分送出後，系統可自動引導委員觀看即時整合成績榜單。
5. **即時監控與刪除**：管理者可在儀表板即時觀看評分進度，所有事件也可於列表中隨時刪除作廢。
6. **匯出報表**：所有人評分完畢後，點擊「下載 Excel 報表」即可存取最終名次與個別詳細分數。

## ⚠️ 管理員權限說明
本平台具有安全存取控管：
- **初始管理員與自動部署**：專案已搭載部署時自動初始化機制。在首次部署至如 Render 的雲端伺服器時，系統將自動配置主要的 `admin` 帳戶。詳細登入資訊應由系統建置人員私下提供給維護人員。若於本地開發，亦支援使用 `python manage.py createsuperuser` 自行建立。
- 所有管理相關頁面（建立、圖表統計、刪除、匯出）皆受到 `@login_required` 安全防護。
